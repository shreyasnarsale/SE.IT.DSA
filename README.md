**Course Code:** 214447  
**Second Year – Information Technology (2019 Course)**  
Savitribai Phule Pune University, Pune  

---

## 📌 Prerequisite
- Fundamental knowledge of Programming Language  
- Basics of Algorithms  

---

## 🎯 Course Objectives

1. Study data structures and their implementation & applications.  
2. Learn searching and sorting techniques.  
3. Understand advanced data structures (Trees, Graphs, Tables).  
4. Learn file organization techniques.  
5. Develop and analyze algorithms.  

---

## 🎓 Course Outcomes

- **CO1:** Analyze algorithms and determine correctness & time complexity.  
- **CO2:** Implement ADTs and data structures.  
- **CO3:** Design algorithms using brute force, divide & conquer, greedy techniques.  
- **CO4:** Solve problems using data structures & algorithmic techniques.  
- **CO5:** Analyze time and space complexity.

- C++  
- 64-bit Open Source Environment

# 📑 List of Laboratory Assignments

---

### 1️⃣ Searching and Sorting  

- Student Database (Array of Structures)  
- Bubble Sort / Insertion Sort  
- Quick Sort (Top 10 Toppers)  
- Search by SGPA  
- Binary Search (Non-recursive)  

---

### 2️⃣ Stack  

- Stack using Singly Linked List  
- Infix → Postfix / Prefix Conversion  
- Postfix / Prefix Evaluation  

---

### 3️⃣ Circular Queue  

- Implementation using Array  
- Enqueue  
- Dequeue  
- Display  
- Handle Queue Full Condition  

---

### 4️⃣ Expression Tree  

- Construct from Postfix / Prefix  
- Recursive & Non-Recursive Traversals  

---

### 5️⃣ Binary Search Tree  

- Insert (Handle Duplicates)  
- Delete  
- Search  
- Traversals  
- Depth  
- Mirror Image  
- Copy Tree  
- Display Parent & Child Nodes  
- Display Leaf Nodes  
- Level-wise Display  

---

### 6️⃣ Threaded Binary Tree  

- In-order Threaded Tree  
- In-order & Pre-order Traversal  

---

### 7️⃣ Graph – Minimum Spanning Tree  

- Graph Representation (Adjacency Matrix / List)  
- Kruskal’s Algorithm  
- Prim’s Algorithm  

---

### 8️⃣ Graph – Shortest Path  

- Dijkstra’s Algorithm  
- Single Source to All Destinations  

---

### 9️⃣ Heap Sort  

- Max Heap / Min Heap  
- Heap Sort Implementation  

---

### 🔟 File Handling  

- Student Database using Sequential File  
- Add Record  
- Delete Record  
- Search Record  
- Display Student Details
